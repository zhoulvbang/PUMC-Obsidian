首先打开 Windows 中安装的 Linux
1. 新建一个目录：`mkdir test`
2. 指定目录/路径：`cd test/`
3. 通过 `ls` 查看该路径下有什么文件
4. 或使用 `ls -a` 查看该路径下文件
5. 使用 `cp`命令拷贝文件，例如：`cp ~/Downloads/* . -v` ；其中 `*` 是通配符，会将所有文件选中；`.` 表示拷贝到当前目录；`-v` 表示会将拷贝信息打印出来
6. 再使用 `ls -l` 以更详细的方式将文件信息列举出来
7. 创建一个子目录：`mkdir -v sub`
8. 将特定格式的文件复制一份到子目录：`cp -av *.gz sub/`
